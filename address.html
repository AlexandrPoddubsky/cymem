cymem.cymem.Address¶

class cymem.cymem.Address¶

A block of number * size-bytes of 0-initialized memory, tied to a Python ref-counted object. When the object is garbage collected, the memory is freed.

Parameters:
  • number (size_t) – The number of elements in the memory block.
  • elem_size (size_t) – The size of each element.
ptr¶

void*

Pointer to the memory block.

addr¶

size_t

Read-only size_t cast of the pointer.
